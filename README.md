# 1. 模块目标


AI Coach用于分析用户历史交易。


输入：

- 交易记录
- K线环境
- 用户交易规则
- 盈亏结果


输出：

- 交易评分
- 优点
- 缺点
- 改进建议
- 训练方向


---

# 2. AI定位


AI不是：

❌ 自动喊单

❌ 预测未来价格

❌ 自动交易


AI是：

✅ 交易复盘老师

✅ 纪律检查员

✅ 策略优化助手

逆趋势：

+10

30分

检查：

是否靠近：

5.3 风险管理

30分

检查：

盈亏比

标准：

=2

1-2

<1

较差

5.4 执行纪律

10分

8. 规则训练模式

9. 错误分类系统

AI自动分类：

特点：

逆势交易

止损过大

特点：

风险超过设定。

盈亏比不足

特点：

赚小亏大。

提前离场

特点：

没有执行目标。
